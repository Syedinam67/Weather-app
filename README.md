## Installation

[node.js](https://nodejs.org/) is required to get `npm` .

If you would like to download the code and try it for yourself:

1. Clone the repo: `https://github.com/YOUR_USERNAME/atmos-weather-app.git`
2. `cd atmos-weather-app`
3. Install packages: `npm install`
4. Set up env: Create `.env` and add `VITE_WEATHER_API_KEY`
5. Build project and launch: `npm run dev`
6. Open your browser at: `http://localhost:5173`
